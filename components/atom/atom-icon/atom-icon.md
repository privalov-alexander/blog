# BaseIcon

> Компонент используется для вывода иконок в проекте.

## Настройка

-   Файл с самими иконоками по дефолту должен лежать по пути `/assets/svg/icon.svg`
-   Файл должен быть написан как спрайт (один `svg` содержащий много `symbol` помеченые через `id`)
-   Названия всех допустимых названий иконок необходимо прописать в переменную `iconTypesAsConst`

> В файле указан дефолтный размер иконок, которые перебиваются через пропс классов

## Использование

```vue
<BaseIcon name="error" />

<BaseIcon name="error" class="size-6" />
```
